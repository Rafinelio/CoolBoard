Apple Silicon cooling control

Monitor thermal sensors and control detected fans from a quiet macOS panel.

Download .pkg View source

Why download

See heat, fan RPM, and cooling state in one focused panel.

CoolBoard is built for Apple Silicon users running local AI, long builds, exports, or other sustained workloads. It makes sensor data readable, exposes detected fans, and keeps manual cooling simple enough to trust.

01 Thermal board CPU, GPU, battery, AirPort, and power sensors in one view
02 Fan control Auto mode or simple 0-100% presets for detected fans
03 Fail-safe Auto Restores system cooling on quit, sleep, wake, and write errors
04 Local source MIT SwiftUI app with no cloud runtime or subscription layer

Install

macOS 14+, Apple Silicon, administrator approval for the helper.

Safety

Auto by default. Manual presets restore on quit, sleep, wake, and errors.

Compatibility

Fanless MacBook Air models run monitoring only and show zero detected fans.

Open Source

MIT licensed SwiftUI app with documented AppleSMC limitations.
